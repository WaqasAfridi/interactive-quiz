# interactive-quiz

A simple Interactive Quiz Website built with HTML, CSS and vanilla JavaScript.

## Features
- Landing page with Start Quiz button
- 7 multiple-choice questions (stored in `questions.js`)
- One question at a time, with Previous / Next / Submit
- Progress bar showing answered count
- Result page with date/time, answered/correct counts, accuracy donut, and per-question feedback
- Restart Quiz
- Responsive design and simple UI animations

## How to run
1. Clone the repo: `git clone https://github.com/WaqasAfridi/interactive-quiz.git`  
2. Open `index.html` in your browser (no build step required).

## Tech
- HTML, CSS, JavaScript
- dayjs for formatted date/time
